测试说明

  1. 确保已安装并启用"智能文本助手"Chrome插件
  2. 在下方文本中选中任意内容
  3. 观察悬浮面板是否正常显示
  4. 测试各个功能按钮是否正常工作
  5. 检查API配置是否正确（需要有效的Kimi API Key）
  6. 重要： 如果结果一闪而过，请检查浏览器控制台是否有错误信息

🔧 调试信息

如果遇到问题，请：

  1. 按F12打开开发者工具
  2. 查看Console标签页的错误信息
  3. 检查Network标签页的API请求状态
  4. 确认插件已正确加载（在chrome://extensions/页面检查）

⚡ 性能测试
